Simple parser
=============

Project works on Python 3.6+

Example usage:
```
python3.6 -m pip install -r requirements.txt
python3.6 crawler.py --help
python3.6 crawler.py http://qwe.ru --regex "<title>.*</title>" --output-file "output.txt" --max-count 10
```
